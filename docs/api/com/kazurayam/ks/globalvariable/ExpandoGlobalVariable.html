        • static java.util.SortedSet<java.lang.String> keySetOfGlobalVariables()

          inspect the 'internal.GlobalVariable' object to find the GlobalVariables contained, return the list of their names. The list will include 2 types of GlobalVariables. 1. GlobalVariables statically defined in the Execution Profile which was applied to this time of Test Case run where the JUnit TestRunner runs. In this category, there are 2 types of GlobalVariable. a) GlobalVariables defined in the Profile actually applied to the test case execution. b) GlobalVariables defined in any of Profiles which are NOT applied to the test case execution. The a) type of GlobalVariable will have some meaningful value. But the b) type will have 'null' value. 2. GlobalVariables dynamically added into the ExpandoGlobalVariable by calling ExpandoGlobalVariable.addGlobalVariable(name,value)
